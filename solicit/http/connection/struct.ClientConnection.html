A struct implementing the client side of an HTTP/2 connection.

It builds on top of an HttpConnection and provides additional methods that are only used by clients.

Fields

session

The Session associated with this connection. It is essentially a set of callbacks that are triggered by the connection when different states in the HTTP/2 communication arise.
